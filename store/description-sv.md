# Butiksbeskrivning (Svenska)

Stjärnmarkera event under Almedalsveckan och exportera ditt personliga schema som en ICS-kalenderfil.

## Beskrivning

Almedalsstjärnan gör det enkelt att planera din Almedalsvecka. Bläddra bland eventen på det officiella programmet på almedalsveckan.info och stjärnmarkera de du vill gå på — direkt i webbläsaren.

**Funktioner:**

- ⭐ Stjärnmarkera event direkt på almedalsveckan.info
- 📋 Se alla dina markerade event i en samlad vy
- 📅 Exportera ditt schema som ICS-fil till valfri kalenderapp
- 🔍 Sortera och filtrera dina markerade event
- 🔒 All data sparas lokalt på din enhet — inget konto krävs

Perfekt för dig som besöker Almedalsveckan i Visby och vill ha koll på ditt personliga schema utan att behöva skapa konton eller dela data med tredje part.
