# Privacy Policy — Almedalsstjärnan

## Data Storage

Almedalsstjärnan stores data exclusively using `chrome.storage.local` on the user's device. All starred events and user preferences remain on-device and are never transmitted elsewhere.

## Data Collection

This extension does **not** collect any user data.

## Data Transmission

This extension does **not** transmit data to any external server. All operations happen locally in the browser.

## Analytics and Tracking

This extension does **not** use analytics, telemetry, or tracking of any kind.
