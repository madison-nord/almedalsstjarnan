## [1.0.0] - 2026-06-21

### Added

- Star and unstar events on almedalsveckan.info program pages
- Bulk star with progress indicator for starring multiple events at once
- ICS calendar export of starred events
- Popup UI for viewing and managing starred events
- Stars page with search, sort, and grid view
- Help/onboarding modal for new users
- Language toggle supporting Swedish and English
- Undo support for accidental star/unstar actions
- Shadow DOM content script isolation to avoid style conflicts with host page
- CI pipeline with lint, typecheck, unit, property, and e2e tests
